# Architecture Decision Record (ADR)

## ADR-001
Use Next.js App Router.
Reason: Modern routing, metadata API, server rendering.

## ADR-002
Use TypeScript.
Reason: Maintainability and scalability.

## ADR-003
Use Tailwind CSS.
Reason: Fast, consistent UI development.

## ADR-004
Use shadcn/ui.
Reason: Accessible reusable components.

## ADR-005
Use Framer Motion + GSAP.
Reason: Premium enterprise animations.

## ADR-006
Use Open SEO.
Reason: Strong metadata and search optimization.

## ADR-007
Deploy on Vercel.
Reason: Excellent Next.js support and global edge deployment.

## ADR-008
Design for future CMS, ERP and multilingual expansion.
